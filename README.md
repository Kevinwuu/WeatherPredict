
# 降雨量預測

## 作法
用BeautifulSoup()爬取網頁資料，拿過去的天氣資料與降雨量做training，透過regression求出回歸直線，再
用爬蟲抓取及時資料做testing，驗證準確率。

## 使用工具
Python 3.5.3

BeautifulSoup

pandas

sklearn.linear_model

...

## 進度
craw_data : 爬取日報表資料。

craw_rain : 爬取即時降雨機率。

craw_year : 爬取年報表資料。

regression : 讀取資料畫出相關性，求regression方程式。
